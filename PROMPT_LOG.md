# Лог работы с ИИ (Prompt Log)

1. **Генерация архитектуры:** Использовался промпт для создания структуры FastAPI и моделей SQLAlchemy (Врачи, Пациенты, Записи).
2. **Реализация безопасности:** Запрос на добавление JWT-аутентификации и защиты эндпоинтов.
3. **Бизнес-логика:** Промпт на добавление проверки занятости врача (валидация времени).
4. **Тестирование:** Система промптов для генерации pytest сценариев с покрытием > 90%.
5. **CI/CD:** Настройка автоматизации через GitHub Actions для проверки кода при пуше.
