# Отчет Code Review

1. **ИИ сгенерировал:** `datetime.now()` без таймзоны. 
   **Проблема:** Ошибки времени при разных серверах. **Исправил:** `datetime.utcnow()`.
2. **ИИ сгенерировал:** Запись в БД без проверки существования врача. 
   **Проблема:** `ForeignKeyViolation`. **Исправил:** Добавлена валидация ID перед сохранением.
3. **ИИ сгенерировал:** Прямой доступ к медкартам по ID. 
   **Проблема:** Нарушение приватности данных. **Исправил:** Добавлена проверка JWT владельца.
4. **ИИ сгенерировал:** Отсутствие лимита в GET запросах. 
   **Проблема:** Падение при 1000+ записях. **Исправил:** Добавлен пагинатор.
5. **ИИ сгенерировал:** Секретный ключ в коде. 
   **Проблема:** Уязвимость. **Исправил:** Вынес в константу (в реале в .env).
