# Отчет по Code Review (Lab 12)
## Вариант 25: Система управления (Врачи)

1. **Проблема:** SQL Injection. **Решение:** Использование SQLAlchemy ORM параметров.
2. **Проблема:** Отсутствие валидации даты. **Решение:** Добавлена проверка на занятость врача.
3. **Проблема:** N+1 Query. **Решение:** Планируется использование joinedload в будущем.
4. **Проблема:** Race Condition. **Решение:** Добавление уникального индекса на время приема.
5. **Проблема:** Plaintext IDs. **Решение:** В реальной системе стоит заменить Integer ID на UUID.
